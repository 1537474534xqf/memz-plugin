# MEMZ-Plugin

## 📋介绍

Yunzai系列Bot插件,本插件基于TRSS-Yunzai作为开发环境,其他框架可能有不支持的地方,可提issue适配

## 安装插件

### 安装本体命令

```bash
git clone --depth=1 https://gitee.com/memzjs/memz-plugin.git ./plugins/memz-plugin/
```

### 安装依赖命令

```bash
pnpm i
```

## 使用

发送 `#MEMZ帮助`查看插件部分功能
