# MEMZ-Plugin

## 介绍

Yunzai系列Bot插件

## 安装插件

### 安装本体命令
```
    git clone --depth=1 https://gitee.com/memzjs/memz-plugin.git ./plugins/memz-plugin/
```
### 安装依赖命令
```
    npm install -P #or pnpm install -P
```
## 使用

发送`#MEMZ帮助`查看插件部分功能
